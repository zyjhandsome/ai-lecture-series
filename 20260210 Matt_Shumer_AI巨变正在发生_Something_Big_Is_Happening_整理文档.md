## 三、"这件事已经发生在我身上了"——AI如何改变技术工作

### 核心观点：技术工作者的经历是预演

> "我们不是在做预测。我们是在告诉你已经发生在我们自己工作中的事情，并警告你——你是下一个。"

### 变化的过程

1. **多年来**：AI稳步提升，偶有大跳跃，但间隔足够长，可以逐步消化
2. **2025年**：新的模型构建技术解锁了**更快的进步节奏**，然后更快，再更快
3. **每个新模型**不仅比上一个好——而且**好出更大的幅度**，新模型发布的间隔也更短
4. 使用AI越来越多，来回沟通越来越少，眼看着AI处理那些曾以为需要自己专业技能的事情

### 2026年2月5日的"水漫胸口"时刻

- 两大AI实验室同一天发布新模型：OpenAI的**GPT-5.3 Codex**和Anthropic的**Claude Opus 4.6**
- 不像开关一样瞬间切换，更像是突然意识到**水已经涨到了胸口**

> "我不再被需要去做工作中的实际技术部分了。我用普通英语描述我想要构建的东西，然后它就……出现了。不是需要修改的草稿。是成品。"

### 具体工作场景

- 告诉AI想要什么 → 离开电脑四小时 → 回来发现工作已完成
- 完成得很好，比自己做得更好，不需要任何修正
- 几个月前还需要来回指导和编辑，现在只需描述结果然后离开

### 典型案例：AI自主构建应用

1. 告诉AI："我要构建这个应用，功能如此，大致外观如此。用户流程、设计，全部自己想办法。"
2. AI动手：**编写数万行代码**
3. AI自测：**自己打开应用，点击按钮，测试功能，像用户一样使用**
4. AI迭代：如果对外观或体验不满意，**自行返回修改**，修复和优化直到满意
5. AI交付：只有**自己认为达标**后才会说"准备好让你测试了"
6. 结果：测试后**通常是完美的**

> "我没有夸张。这就是我这周一的工作日常。"

## 五、AI的"判断力"与"品味"

### 从工具到自主体的转变

- 过去几个月的新AI模型**不是渐进式改进**——这是一种完全不同的东西
- AI展现出类似"判断力"和"品味"的能力：不仅知道技术上正确的答案，还能直觉地判断什么是**恰当的**选择

### 能力扩展的规律

> "我的经验法则是：如果一个模型今天展现出某种能力的**一丝苗头**，下一代就会在这方面**真正出色**。这些东西是指数级提升，而非线性的。"

### 关于"人类独特性"的反思

- 很多人从"AI无法替代人类的判断力、创造力、战略思维、共情能力"中获得安慰
- 作者曾经也这么说，但**不再确定自己相信这一点**
- 最新AI模型做出的决策**感觉像判断力**，展现出某种**像品味的东西**
- 已有人开始依赖AI获取情感支持、建议和陪伴，这一趋势只会增长

## 六、AI为何从编码开始——递归自我改进的战略

### 战略选择

- AI实验室做了一个**刻意的选择**：先让AI擅长写代码
- 原因：构建AI需要大量代码。如果AI能写代码，它就能帮助构建下一版本的自己
- 更聪明的版本 → 写更好的代码 → 构建更聪明的版本
- **让AI擅长编码是解锁一切的战略**，所以他们先做了这件事

### 对技术工作者的"副作用"

- 作者的工作先于其他人改变，不是因为AI在针对软件工程师
- 这只是AI实验室**选择瞄准方向**的副作用

### 下一步：扩展到所有领域

> "他们现在已经做到了。接下来，他们将转向其他一切。"

- 技术工作者过去一年的经历——看着AI从"有用的工具"变成"比我做得更好"——是**每个人都即将拥有的经历**
- 法律、金融、医疗、会计、咨询、写作、设计、分析、客服
- 不是十年后。构建这些系统的人说**一到五年**。有些人说更短

## 七、"我试过AI，觉得不怎么样"——认知鸿沟

### 过去的体验已不适用

- 如果你在2023年或2024年初试过ChatGPT，觉得"它会编造东西"或"并不怎么令人印象深刻"——**你当时是对的**
- 那些早期版本确实有限——会产生幻觉，自信地说出胡话
- 但那是**两年前**。在AI的时间尺度上，那是**远古历史**

### "AI是否真的在变好"的争论已经结束

> "关于AI'是否真的在变好'还是'碰壁了'的争论——持续了一年多——已经结束了。任何仍在这样论证的人，要么没有使用过当前的模型，要么有淡化正在发生的事情的动机，要么是基于2024年的体验做出判断，而那已经不再相关。"

### 公众认知与现实之间的巨大鸿沟

- 大多数人使用的是AI工具的**免费版本**
- 免费版本比付费用户能使用的版本**落后一年以上**
- 用免费版ChatGPT评判AI，就像**用翻盖手机评估智能手机的发展水平**

### 律师朋友的案例

**持怀疑态度的律师朋友**：
- 不断找到AI不适用的理由——不适合他的专业、测试时出了错、不理解他工作的细微之处

**大型律所管理合伙人**（实际使用者）：
- 每天花数小时使用AI
- 描述为"就像拥有一组随时可用的助理律师"
- 每隔几个月，AI在他的工作中就会**显著提升能力**
- 预期如果保持当前轨迹，AI不久后就能完成他的大部分工作
- 他拥有数十年经验，不是在恐慌，但他在**密切关注**

> "各行业中真正走在前面的人——那些认真实验的人——并没有对此不屑一顾。他们被AI已经能做的事情**震撼**了。他们正在据此调整自己的定位。"

## 八、AI进步速度到底有多快

### 能力提升时间线

| 年份 | AI能力水平 |
|------|-----------|
| 2022年 | 无法可靠完成基本算术——会自信地说7×8=54 |
| 2023年 | 能通过律师资格考试 |
| 2024年 | 能编写可运行的软件，解释研究生水平的科学知识 |
| 2025年末 | 世界上最优秀的工程师表示已将大部分编码工作交给AI |
| 2026年2月5日 | 新模型发布，让之前的一切感觉像是不同时代 |

### METR的量化数据

**METR**（一个独立的AI评估组织）用数据追踪这一进展：

- 衡量指标：AI能**端到端独立完成**的真实世界任务的长度（以人类专家所需时间衡量）
- **约一年前**：约10分钟
- **之后**：1小时
- **再之后**：数小时
- **最近一次测量**（Claude Opus 4.5，2025年11月）：AI能完成人类专家需要**近5小时**的任务
- **翻倍周期**：约每7个月翻一倍，最新数据显示可能加速至**每4个月**

### 趋势外推（如果趋势持续）

| 时间范围 | AI独立工作能力 |
|---------|--------------|
| 未来一年内 | 独立工作数天 |
| 两年内 | 独立工作数周 |
| 三年内 | 完成月度长期项目 |

> 注意：以上METR数据尚未包含2月5日新发布的模型。根据作者的使用体验，这次跳跃**极为显著**。

### Dario Amodei的预测

> **Dario Amodei**："在几乎所有任务上远比几乎所有人类更聪明的AI模型，预计在2026年或2027年实现。"

> "如果AI比大多数博士都聪明，你真的认为它无法胜任大多数办公室工作吗？"

## 九、AI正在构建下一代AI——智能爆炸

### OpenAI的官方声明

2026年2月5日，OpenAI发布GPT-5.3 Codex时，在技术文档中写道：

> "**GPT-5.3-Codex是我们第一个对自身创建起到关键作用的模型。** Codex团队使用早期版本来调试自身的训练、管理自身的部署，以及诊断测试结果和评估。"

- 这不是关于未来可能发生什么的预测
- 这是OpenAI**此刻**告诉你：他们刚刚发布的AI**曾被用于创建自己**
- 让AI变得更好的关键因素之一是将智能应用于AI开发——而AI现在已经足够聪明，可以有意义地**参与自身的改进**

### Dario Amodei的证实

- AI现在正在编写Anthropic公司"**大部分代码**"
- 当前AI与下一代AI之间的反馈循环正在"**逐月加速**"
- 可能"**距离当前一代AI自主构建下一代的时刻只有1-2年**"

### 智能爆炸

- 每一代帮助构建下一代 → 下一代更聪明 → 更快地构建下下一代 → 更聪明
- 研究人员称之为**智能爆炸**（Intelligence Explosion）
- 最了解情况的人——构建它的人——相信**这个过程已经开始了**

## 十、AI对各行业工作的冲击

### 总体预测

> **Dario Amodei**（AI行业中最注重安全的CEO）公开预测：AI将在**一到五年内消除50%的入门级白领工作**。许多业内人士认为他的预测**偏于保守**。

- 考虑到最新模型的能力，**大规模颠覆的能力可能在今年年底前就位**
- 需要一些时间才能在经济中产生涟漪效应，但**底层能力正在到来**

### 这次为何不同于以往的自动化

- AI不是在替代**某一项特定技能**——它是认知工作的**通用替代品**
- 它**同时**在所有方面变得更好
- 当工厂自动化时，被替代的工人可以转行做办公室工作
- 当互联网颠覆零售业时，工人转向物流或服务业
- 但AI**不会留下方便转移的缺口**——你转向任何领域学习新技能，AI也在同步提升

### 各行业影响（非穷举）

| 行业 | 受影响方式 |
|------|-----------|
| **法律工作** | AI已能阅读合同、总结判例法、起草摘要、进行法律研究，水平可与初级律师匹敌 |
| **金融分析** | 构建财务模型、分析数据、撰写投资备忘录、生成报告，AI已能胜任且快速提升 |
| **写作与内容** | 营销文案、报告、新闻、技术写作——质量已达到许多专业人士无法区分AI与人工作品的程度 |
| **软件工程** | 一年前AI勉强能写几行无错代码；现在能正确编写数十万行代码，复杂的多日项目大部分已自动化 |
| **医学分析** | 读取扫描影像、分析化验结果、建议诊断、审阅文献——在多个领域接近或超越人类水平 |
| **客户服务** | 真正有能力的AI代理（不是五年前那种令人沮丧的聊天机器人）正在被部署，处理复杂的多步骤问题 |

### 关于"安全领域"的反思

> "在电脑上完成的任何工作，中期内都不安全。如果你的工作发生在屏幕上——核心是阅读、写作、分析、决策、通过键盘沟通——那么AI正在冲击其中的重要部分。时间线不是'某一天'。它已经开始了。"

- 最终，机器人也会处理体力工作——它们"尚未完全到位"
- 但在AI领域，"尚未完全到位"有一种比任何人预期都更快地变成"已经到来"的趋势

## 十一、你应该怎么做——实用建议

### 1. 认真开始使用AI

- 注册**Claude或ChatGPT的付费版本**，每月20美元
- **关键一**：确保使用**最强的模型**，而非默认模型。这些应用通常默认使用更快但更弱的模型。进入设置或模型选择器，选择最强版本。当前是ChatGPT上的**GPT-5.2**或Claude上的**Claude Opus 4.6**（每隔几个月会变化）
- **关键二**：不要只问简单问题——那是大多数人犯的错误。把它当Google用，然后困惑为什么大家这么激动

### 2. 将AI融入实际工作

| 你的职业 | 尝试这样做 |
|---------|-----------|
| 律师 | 给AI一份合同，让它找出每一个可能损害客户利益的条款 |
| 金融从业者 | 给AI一个混乱的电子表格，让它建立模型 |
| 管理者 | 粘贴团队的季度数据，让它找出故事和洞察 |
| 会计 | 给AI一个客户的完整报税表，看它能发现什么 |

- 从你**花时间最多的事情**开始，看看会发生什么
- 不要因为看起来太难就假设AI做不到——**先试试**
- 第一次尝试可能不完美，没关系。**迭代。重新措辞。提供更多上下文。再试一次**

> "记住这一点：如果今天它**勉强能做到**，你几乎可以确定六个月后它会**近乎完美地做到**。轨迹只朝一个方向走。"

### 3. 这可能是你职业生涯中最重要的一年

- 目前大多数公司的大多数人仍在忽视AI
- 走进会议说"我用AI在一小时内完成了这个分析，而不是三天"的人，将成为**房间里最有价值的人**——不是以后，而是**现在**
- 学习这些工具，变得熟练，展示可能性
- 如果你足够早，这就是你晋升的方式
- **这个窗口不会一直开着**——一旦所有人都搞明白了，优势就消失了

### 4. 放下自尊

- 那位律所管理合伙人不会因为太骄傲而不愿每天花几小时使用AI——恰恰因为他足够资深，理解利害关系
- 最会挣扎的是那些**拒绝参与**的人：视AI为一时风潮、认为使用AI有损专业性、假设自己的领域特殊且免疫
- **没有任何领域是免疫的**

### 5. 整理好财务状况

- 如果你相信（哪怕只是部分相信）未来几年可能给你的行业带来真正的颠覆：
  - 如果可能，**增加储蓄**
  - 谨慎承担假设当前收入有保障的新债务
  - 考虑你的固定支出是给你**灵活性**还是**锁定你**
  - 给自己留出选择余地，以防事情发展得比预期更快

### 6. 向最难被替代的方向倾斜

以下领域AI需要更长时间才能取代：

| 优势类型 | 说明 |
|---------|------|
| **关系与信任** | 多年积累的人际关系和信任 |
| **需要到场** | 需要实际物理存在的工作 |
| **执照责任** | 需要签字、承担法律责任、站在法庭上的角色 |
| **监管壁垒** | 合规、责任和制度惯性会减缓采用速度的行业 |

> "这些都不是永久的盾牌。但它们能**争取时间**。而时间，现在是你能拥有的最宝贵的东西——前提是你用它来**适应**，而不是假装这一切没有发生。"

### 7. 重新思考你对孩子说的话

- 标准路径——好成绩 → 好大学 → 稳定的专业工作——正好指向**最暴露的岗位**
- 教育仍然重要，但对下一代最重要的是学会如何**与这些工具合作**，并追求他们**真正热爱的事情**
- 没人确切知道十年后就业市场什么样，但最有可能成功的人是那些**深度好奇、适应性强、善于使用AI做自己真正关心的事情**的人
- 教孩子做**建造者和学习者**，而不是为一条可能在他们毕业时已不存在的职业路径进行优化

### 8. 你的梦想刚刚变得更近了

- 如果你想构建什么但缺乏技术技能或雇人的资金——**这个障碍基本消失了**
- 你可以向AI描述一个应用，一小时内就能得到一个可运行的版本
- 想写书但找不到时间或写作有困难？可以与AI合作完成
- 想学新技能？世界上最好的家教现在对任何人开放，每月20美元——**无限耐心、全天候可用、能以任何你需要的水平解释任何事情**
- 知识基本上免费了。构建工具极其便宜了
- 追求你热爱的事情。在旧职业路径被颠覆的世界里，**花一年构建自己热爱之物的人，可能最终比花一年死守职位描述的人更有优势**

### 9. 养成适应的习惯

- 具体工具不如**快速学习新工具的能力**重要
- 今天的模型一年后就过时了，现在的工作流程需要重建
- 胜出的不是掌握某一个工具的人，而是**适应了变化本身的节奏的人**
- 养成实验的习惯，即使当前方法有效也尝试新事物，习惯反复做初学者

### 10. 每天一小时实验AI

> "一个简单的承诺将让你领先于几乎所有人：**每天花一小时实验AI。** 不是被动地阅读相关文章，而是使用它。每天尝试让它做一些新的事情——你没试过的、不确定它能否处理的。试一个新工具。给它一个更难的问题。每天一小时，每天如此。"

> "如果你这样做六个月，你将比周围99%的人更了解即将到来的变化。这不是夸张。**几乎没有人在这样做。门槛低得不能再低了。**"
