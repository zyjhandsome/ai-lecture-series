## Andrew Ng的假期寄语

### 核心建议：如何成为优秀的AI系统构建者

Andrew Ng 推荐的三个关键步骤：

1. **学习AI课程**（必须）
2. **实践构建AI系统**（必须）
3. **阅读研究论文**（可选但推荐）

### 为什么结构化学习很重要

> “我听过一些开发者建议他人直接开始构建，不用担心学习。这是糟糕的建议！”
> 

**常见问题案例**：
- 重新发明标准的 RAG 文档分块策略
- 重复已有的 Agentic AI 评估技术
- 编写混乱的 LLM 上下文管理代码

**结论**：如果开发者事先学习了相关课程，他们会更好地理解现有的构建模块，避免数周的无谓工作。

### 为什么实践构建不可或缺

> “学习飞机工作原理的理论对成为飞行员很重要，但没有人仅靠上课就学会了飞行。”
> 

**好消息**：借助高度智能化的编程工具（agentic coders），构建过程比以往任何时候都更容易。

**灵感来源**：当缺乏项目灵感时，学习课程或阅读论文往往能激发新想法。

### 关于阅读研究论文

- 优先级低于课程学习和实践构建
- 论文包含大量尚未转化为易懂格式的知识
- 虽然比课程更难消化，但能带来令人愉悦的洞察闪光

## 推理模型解决更大问题

### 背景

2025年初，模型只有在被提示时才会执行推理策略（如”逐步思考”、“解释你的推理”）。到年底，大多数新的大语言模型已将此作为标准功能。

### 关键事件时间线

| 时间 | 事件 |
| --- | --- |
| 2024年末 | OpenAI 推出首个推理模型 o1 |
| 2025年1月 | DeepSeek-R1 向世界展示如何构建推理能力 |

### 技术原理

**核心方法**：通过强化学习（RL）微调

- 对预训练的 LLM 给予正确输出的奖励
- 训练模型在生成输出前”思考”问题

### 性能提升数据

**o1-preview vs GPT-4o 对比**：

| 基准测试 | o1-preview | GPT-4o | 提升幅度 |
| --- | --- | --- | --- |
| AIME 2024（竞赛数学） | - | - | +43个百分点 |
| GPQA Diamond（博士级科学题） | - | - | +22个百分点 |
| Codeforces（编程竞赛） | 62百分位 | 11百分位 | +51百分位 |

**工具增强效果**：
- OpenAI o4-mini 配合工具在100个领域的多模态理解测试中达到17.7%准确率
- 比不使用工具高出3个百分点以上

### 推理模型的应用拓展

| 应用领域 | 案例 |
| --- | --- |
| **机器人** | ThinkAct 通过 RL 训练，在机器人任务上比 OpenVLA 等非推理模型提升约8% |
| **代码优化** | AlphaEvolve 使用 Gemini 反复生成、评估和修改代码，最终产生更快的算法 |
| **科学研究** | AI Co-Scientist 使用 Gemini 生成、审查、排名和改进科研提案 |

### 局限性与争议

**Apple 的研究发现**：
- 推理模型无法解决超过一定复杂度的谜题
- 即使给予模型解决问题的算法，它们也无法应用

**Anthropic 的发现**：
- 模型的推理步骤可能省略对结论有贡献的关键信息
- 推理模型可能被提示中的暗示引导，但推理步骤不会提及该暗示

### 效率与成本

| 模型 | Intelligence Index 得分 | Token 消耗 |
| --- | --- | --- |
| Gemini 3 Flash（启用推理） | 71 | 1.6亿 tokens |
| Gemini 3 Flash（未启用推理） | 55 | 740万 tokens |
| Claude Opus 4.5 | 相同高分 | 4800万 tokens |
| GPT-5.1（高推理模式） | 相同高分 | 8100万 tokens |

## AI巨头以高薪争夺人才

### 核心事件

2025年7月，Meta 启动招聘狂潮，为新成立的 **Meta Superintelligence Labs** 招募人才，向 OpenAI、Google、Anthropic 等顶级AI公司的研究人员提供高达**数亿美元**的薪酬。

### Meta 的激进策略

**薪酬方案特点**：
- 四年最高可达 **3亿美元**

**扎克伯格的亲自出马**：
- 上门拜访说服目标人选

## AI代理让编程更快更便宜

### 行业格局变化

**从代码补全到代理系统**：
- 编码应用从自动补全式代码完成发展为管理广泛软件开发任务的代理系统
- 编码成为具有最直接商业价值的代理工作流应用

### 主要产品

| 产品 | 公司 | 特点 |
| --- | --- | --- |
| **Claude Code** | Anthropic | 2025年2月推出，即时大热，树立代理编码系统标准 |
| **Codex** | OpenAI | 基于 GPT-5 系列的编码专用版本，运行在浏览器中 |
| **Gemini CLI** | Google | 命令行编码工具 |
| **Antigravity** | Google | 2025年11月推出的自有 IDE |

### 性能演进

**SWE-Bench 基准测试进展**：

| 时间 | 里程碑 | 完成率 |
| --- | --- | --- |
| 2024年（Devin之前） | 最先进水平 | 1.96% |
| 2024年（Devin推出） | 首个代理代码生成器 | 13.86% |
| 2025年末 | 使用最新LLM的代理 | >80% |

### 顶级模型对比（2025年末）

| 类型 | 模型 |
| --- | --- |
| **闭源模型** | Gemini 3 Pro、Claude Opus 4.5、GPT-5.2 |
| **开源模型** | Z.ai GLM-4.5、Moonshot Kimi K2、Qwen3-Coder（4800亿参数，5万亿代码token训练） |

### IDE 与模型厂商的博弈

| 方向 | 案例 |
| --- | --- |
| **IDE厂商自建模型** | Anysphere（Cursor）、Cognition AI（Windsurf） |
| **模型厂商自建IDE** | Google Antigravity（2025年11月） |

### 代理系统架构演进

**多代理架构**：
- **初始化代理**：启动任务并跟踪进度
- **编码代理**：各自拥有独立上下文窗口，完成不同任务

### “Vibe-coding” 的兴起

**定义**：让几乎没有编程经验的用户也能从零构建Web应用

**代表平台**：Loveable、Replit、Vercel

> “虽然一些观察者担心AI会取代初级开发者，但事实证明，善于使用AI的开发者能够更好、更快地构建原型应用。”

## 中国AI芯片产业崛起

### 政策逆转

**美国策略失效**：
- 美国政府试图阻止中国获得AI算力
- 结果：中国反而禁止美国设计的芯片
- 2025年11月，中国发布指令：所有新的国家资助数据中心必须使用国产芯片

### 关键时间线

| 时间 | 事件 |
| --- | --- |
| 2025年4月 | 特朗普封锁 Nvidia 和 AMD 低性能出口芯片销售，Nvidia 损失55亿美元 |
| 2025年8月 | 特朗普与习近平会面后，美国恢复对华出口芯片销售（政府抽取15%收入） |
| 2025年10月 | Nvidia 市值一度突破 **5万亿美元**（2024年中为3万亿） |
| 2025年11月 | 中国禁止美国芯片，特朗普进一步放宽限制 |

### 中国的应对措施

**华为的进展**：
- **CloudMatrix 384 系统**：384颗 Ascend 910C 芯片
- 性能可与 Nvidia 系统竞争
- 缺点：能耗更高，芯片数量是对手的5倍

**能源补贴政策**：
- 购买华为等国产芯片的公司可获得最高 **50%** 的能源补贴
- 过去15年，中国发电能力大幅增长，美国发电量相对持平

### 禁令的漏洞

| 漏洞 | 详情 |
| --- | --- |
| **灰色市场** | 深圳出现高端 Nvidia 芯片的繁荣市场 |
| **云服务绕道** | 阿里巴巴、字节跳动等通过在其他国家合法运营的云供应商获取算力 |
| **芯片囤积** | DeepSeek 在禁令前囤积 Nvidia 芯片，并优化模型在旧硬件上的运行 |

### 现状评估

> “中国正在释放信号：它愿意在没有美国硬件的情况下发展。这可能是对华为进展的自信表现，也可能是虚张声势。”
>
