## xAI 成立两年半的成就回顾

马斯克在开场中回顾了 xAI 成立仅两年半以来取得的成就，强调公司虽然只是一个"蹒跚学步的幼儿"，但已在多个领域取得了领先地位，而竞争对手们有的已经运营了5年、10年甚至20年。

### 主要成就

- **语音领域**：Grok 语音功能达到行业第一
- **图像与视频生成**：生成的图像和视频数量超过所有竞争对手总和
- **预测能力**：Grok 420 预测模型在所有AI预测排名中夺冠——预测是智能的关键指标之一
- **多项排行榜榜首**：在多个AI基准测试中名列前茅
- **Grok 核心应用**：推出了出色的应用产品
- **X 应用改进**：对 X 应用进行了根本性改进
- **Grokipedia**：正在打造超越维基百科的百科全书，目标是涵盖视频和图像数据的"银河百科全书"
- **算力里程碑**：率先实现10万台 H100 GPU 训练集群，即将实现100万台 H100 等效 GPU 训练能力

> **马斯克**："对于任何科技公司的竞争力而言，重要的不是某一时间点的位置，而是你的速度和加速度。如果你在任何技术领域比其他人移动得更快，你就会成为领导者。xAI 的速度比任何其他公司都快，没有人能与之匹敌。"

- 目标是让 Grok 在工程、法律、医学等所有领域都**真正有用**且**答案可靠**

### 产品端展望

> **产品负责人**："在接下来几个月内，我们将看到知识工作者的产出增加十倍。我们正在建造一个门户，让你能够完成所有工作，放大每个人的能力，让他们实现远超独自所能完成的成就。"

### 工作文化

## 编码团队（Coding）

### 编码模型的质变

MRO（Micro）领导编码团队，他描述了编码模型从"不可用"到"真正有用"的转变：

> **MRO**："最近编码模型发生了质的飞跃。以前你需要像牵着一个蹒跚学步的孩子一样去引导AI做出改变。而现在当我描述一个问题时，我只需要像对一个已经熟悉代码库的同事工程师一样表述就够了。这是一个巨大的变化。"

### 编码模型的能力

- 不仅能**编写代码**，还能**调试代码**
- xAI 持续运行 Grok Code 数小时，确保对训练系统的复杂更改在生产环境中正常工作
- 编码模型让工程师**提高10倍生产力**

### 递归自我改进

> **MRO**："这不仅仅是加速我们自己写代码。我们正走在递归自我改进的道路上——当前一代 Grok Code 正在训练下一代 Grok Code。我们正处于指数级腾飞的轨道上。"

### 编码的未来

Gorang 补充道：

> **Gorang**："我们在编码领域正走向奇点（Singularity）。所以我们决定让公司最好的工程师 Micro 来领导编码工作。"

马斯克做出了更激进的预测：

> **马斯克**："我认为到今年年底，事情可能会发展到你甚至不需要编码了——AI 直接创建二进制文件。AI 可以创建比任何编译器更高效的二进制文件。你只需说'为这个特定目标创建优化的二进制文件'，实际上你就绕过了传统编码。这是一个不需要的中间步骤。"

马斯克还表示，**Grok Code 将在2-3个月内达到业界最先进水平**。

## MacroHard 项目：数字化公司模拟

### 项目概述

Toby Pohlen 领导的 MacroHard 项目（名字是对微软 Microsoft 的戏谑）旨在构建一个完全能力的、实时的人类模拟器。

> **Toby Pohlen**："给人类配备电脑是个好主意，所以我们对AI也做同样的事——就像电影《盗梦空间》一样，我们给电脑配电脑。MacroHard 正在构建一个完全可用的数字化实时人类模拟器，能够在电脑上做任何人类能做的事情，包括使用工程和医学领域的高级工具。应该有完全由AI设计的火箭引擎。"

### 从 CLI 到 GUI

John 负责强推理模型，介绍了从命令行到图形界面的扩展：

> **John**："我们正在构建强推理模型来控制CLI（命令行界面）。但世界上80-90%甚至95%的软件都有GUI（图形用户界面）。要真正让人们的生活更轻松，我们需要开发能够解决日常GUI任务的模型。MacroHard 将实现跨桌面的端到端编排，带来巨大的经济繁荣。"

### 马斯克的评价

> **马斯克**："MacroHard 项目随着时间的推移实际上可能会成为我们最重要的项目。我们谈论的是模拟整个人类公司。世界上最有价值的公司，它们的输出都是数字化的——它们实际上不生产硬件。因此，应该有可能完全模拟任何数字输出的公司。这将迎来一个我们目前难以想象的繁荣时代。"

> **马斯克**："这就是为什么'MacroHard'这几个字被画在训练集群的屋顶上——因为那就是它要建造的东西。"

### 招聘标准

John 提出了 MacroHard 的三个招聘标准：

1. **聪明**：能否解决困难问题？
2. **有驱动力**：有野心吗？想要赢吗？
3. **人品好**：你愿意和他们一起工作吗？

## 基础设施与工程团队

### ML Infra / 预训练团队

团队成员描述了训练 Grok 3 时的经历：

> **ML Infra 团队成员**："当我们训练 Grok 3 时，我们为此构建了预训练框架。我们有10万台 H100，它们刚刚交付，但软件还没准备好。在3万台规模时，我们意识到软件实际上无法正常工作。我们几乎重写了一半的软件，因为在数据中心里有太多你无法预料的事情——交换机在抖动、链路在波动、GPU在烧毁、你有数值精度问题。你需要让10万台 H100 同步运行。一个训练步骤大约5秒，在这5秒内，一切都可能发生。你需要编写一个能在这些情况下继续前进的系统。"

核心特点：
- 预训练团队总共约**15人**，其中约**7人**负责实际训练系统
- 保持极高的**人才密度**

### RL 和推理团队

Lemi（Leon Min）负责大规模强化学习训练和生产推理：

> **Lemi**："我们在地球上运行大规模强化学习训练和生产推理——可能很快也会在太空中运行。我们正在构建从10万芯片扩展到百万芯片的系统。我们优化堆栈的每个方面——并行性、预填充、解码，并使其对每种已知和未知的硬件故障具有韧性。"

### Jax 团队

Yulom 领导 Jax 团队，负责 Jax GPU 优化：

> **Yulom**："在 xAI，Jax 团队只有几位工程师，负责优化超大规模GPU训练。即使在大规模上运行 Hello World 也可能很复杂。我们负责支持整个公司——从预训练、基础模型、强化学习到多模态——从1万、10万到可能100万台 H100 等效GPU的规模。我们需要定制整个 Jax 堆栈，包括编译器和运行时。"

### 内核团队

Prangul 领导内核团队：

> **Prangul**："内核团队位于训练和推理堆栈的最底层。我们的代码运行在百万等效GPU内部。GPU内部有数十万个线程，这些线程互相通信以进行矩阵乘法、计算注意力分数，有些甚至需要与其他百万台GPU通信。我们热衷于优化每一微秒的性能，榨干这些GPU的每一滴性能。"

### 工具链团队

Ashtip 负责工具链团队：

> **Ashtip**："作为工具链团队，我们负责构建平台、框架和基础设施，供人类和AI代理使用我们的产品。我们从构建人类数据平台开始，扩展到构建内部工程平台，通过它运行部署、评估和查看训练结果。"

### 核心产品基础设施与 API

Toby Pohlen（同时负责 MacroHard）兼管核心产品基础设施：

> **Toby**："任何时候你使用我们的产品——grok.com、API认证、status.x.ai——这都是核心产品基础设施团队的工作。我们在伦敦有大量团队成员。我们在每天高峰时段下午4点保持系统运行，晚上系统出问题时我们会被叫醒。如果你对解决复杂的分布式问题和处理混乱数据感兴趣，这就是你应该加入的团队。"

- **真实性追求（Truth Seeking）**方面的提升
- **减少政治偏见**
- 回应更加**清晰有力**

## Memphis 超级计算集群

### 数据中心概况

Hiner 和 Dan/Spencer 从 Memphis 数据中心现场连线：

## 太空愿景与星际扩展

马斯克在大会结尾描绘了 xAI 与 SpaceX 合并后的宏大太空愿景。

### 核心理念

> **马斯克**："要理解宇宙，你必须探索宇宙。仅仅在地球上用望远镜和对撞机能学到的东西是有限的。你最终必须走出去，探索宇宙才能理解它。这就是 SpaceX 和 xAI 合并背后的动机——加速人类理解宇宙和将意识之光延伸到星际的未来。"

### 能源利用的宏观视角

| 层级 | 能源规模 |
|------|---------|
| 当前文明 | 仅使用地球潜在能源的约1% |
| 太阳能的百万分之一 | 约为当前文明使用能源的100万倍 |
| 太阳总质量占比 | 太阳系总质量的99.8% |

> **马斯克**："地球实际上只是一粒漂浮在黑暗中的微小尘埃。你必须超越地球这粒尘埃，才能对利用太阳能源产生任何有意义的影响。"

### 分阶段计划

### 马斯克的终极展望

> **马斯克**："很难想象那个规模的智能会思考什么。但看到它发生将令人无比激动。我真的很想看到月球上的质量驱动器，一个接一个地把AI卫星射入深空。我想不出比这更史诗般的事情了——月球上的质量驱动器和一个自给自足的月球城市，然后超越月球到火星，穿越太阳系，最终去往群星之间，访问所有这些恒星系统。也许我们会遇到外星人。也许我们会发现一些持续了数百万年的文明遗迹。但只有当我们走出去探索时，这一切才有可能发生。"
